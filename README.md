# Telegram Global Search Bot

Bot Telegram untuk mencari channel, grup, pesan, video, gambar, file, link, musik, dan bot secara global.

## Fitur
- Pencarian tanpa command (langsung ketik keyword).
- Navigasi paginasi (10 hasil per halaman).
- Filter kategori menggunakan inline keyboard.
- Kompatibel dengan Railway.

## Teknologi
- Python 3.11
- Telethon (MTProto API)
- Docker (untuk Railway)
